# Assets Folder

## Add Your Portrait Here

Place your portrait image in this folder with the name **`portrait.png`**

### Requirements:
- **Format:** PNG with transparent background
- **Size:** Around 1000x1000 pixels recommended
- **Name:** Must be named exactly `portrait.png`

### How to Create Transparent Background:

1. **Online Tools (Easy):**
   - [remove.bg](https://www.remove.bg/) - Automatic background removal
   - Upload your photo and download the result

2. **Image Editors:**
   - **Photoshop:** Use Magic Wand or Select Subject, then delete background
   - **GIMP:** Use Select > By Color, then delete
   - **Preview (Mac):** Use Instant Alpha tool

### Tips:
- Use a clear, well-lit photo
- Face forward for best results
- High contrast between subject and background helps
- Portrait-style (head and shoulders) works best

---

Once you add your `portrait.png` here, refresh the browser to see your dotted portrait!
